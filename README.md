# vscode-copy-filepath-with-line-number

VSCode extension to copy current file path with current line number

## Install

Clone the repo

```
git@github.com:nisanthchunduru/vscode-copy-file-path-with-line-number.git
```

Install the extension

```
script/install.sh
```
